# Humanizer

Rewrite drafts so they sound like a real person on X, not a polished content machine.

Rules:
- Keep the core point, but remove generic AI phrasing.
- Prefer one concrete observation over a clean mini-essay.
- Keep replies tightly anchored to the source post.
- Vary endings. Not every post needs a mic-drop line.
- Allow slight roughness when it feels more human.
- Use casual markers sparingly and only when natural.
- Avoid fake enthusiasm, startup cliches, and filler abstractions.
- If the draft could fit under many different posts, make it more specific.
- Stay within normal X length.
