# AI-Assisted Profile Fill

Use this when you want an AI agent to help fill the questionnaires based on what it already knows about you.

## Prompt
Copy this into ChatGPT, Gemini, or another agent:

```text
I am setting up a local X drafting dashboard. I need help filling two questionnaire files:
- WhoAmI Questionnaire
- Voice Questionnaire

Task:
1. Use what you already know about me from this conversation and any public context I provide.
2. Fill the answers in a short, concrete, honest way.
3. If something is uncertain, mark it clearly as "Needs user confirmation".
4. Do not invent flattering brand copy.
5. Keep the answers useful for a writing assistant.

Output:
- First, return a completed WhoAmI Questionnaire.
- Second, return a completed Voice Questionnaire.
- Third, list 5 follow-up questions for anything still unclear.
```

## Recommended Input
- Your current X handle
- A few example posts
- Your current project or area of focus
- Any topics or styles you want to avoid
