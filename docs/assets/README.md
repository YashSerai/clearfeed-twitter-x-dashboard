# Asset Placeholders

Add public-facing assets here before posting the repo more widely.

Recommended files:
- `dashboard-screenshot.png`
  Use a clean, sanitized screenshot of the main Clearfeed dashboard.
  Good content: overview row, setup status, queue, and voice review section.
- `clearfeed-workflow.gif`
  Use a 5-10 second GIF showing:
  open candidate -> draft reply -> edit draft -> copy draft -> mark manual.
- `social-preview.png`
  Optional GitHub social preview image for the repo page.
  Recommended size: 1280x640.

Avoid including:
- private account handles you do not want public
- credentials
- personally sensitive drafts
- Telegram chat IDs or bot data
