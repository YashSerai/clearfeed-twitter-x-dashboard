# Launch Checklist

## Repo
- Confirm `.env`, browser session state, logs, and SQLite files are not tracked.
- Add sanitized dashboard screenshots or a short GIF.
- Replace the clone URL placeholder in `README.md`.
- Review profile templates for tone and clarity.

## Product
- Test list discovery with user-provided list URLs only.
- Test copy-and-manual workflow from the dashboard.
- Test Telegram only as an optional mirror, not the main workflow.

## Messaging
- Lead with signal discovery and human-in-the-loop drafting.
- Avoid calling it a bot or growth hack.
- Show one concrete before/after example of a drafted reply.
